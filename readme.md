# 🇸🇦 Saudi Test Data Generator (GUI Tool)

A lightweight, extendable Python tool that generates **realistically formatted fake data** for testing systems, APIs, and UI forms – especially tailored for Saudi use cases.

Built with `Tkinter` for an easy-to-use GUI and designed for software testers and QA professionals.

---

## ✅ Features

- Generate **fake but valid-looking data** for:
  - Saudi National IDs (for Saudis and Residents)
  - IBAN numbers (SA format)
  - Mada Card numbers (with Luhn checksum)
  - STC Pay / Mada / URPay account numbers
  - Hijri date (based on today’s Gregorian date)
  - Currency values in SAR and USD
  - Arabic full names (random combinations)
  - Saudi-style mobile numbers (050-059 prefixes)
- Fully configurable GUI: choose which fields to include
- Export to:
  - `.json` format
  - `.csv` format
- Simple, clean interface built with `Tkinter`
